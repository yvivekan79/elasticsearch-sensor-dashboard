This application provides a complete setup for Elasticsearch data streams with sensor telemetry data:

  • Temperature sensors data (temperature readings)
  • Air quality sensors data (CO, NO2, O3, PM10, PM2.5, SO2)

The application sets up ingest pipelines, index templates, and data streams in Elasticsearch, and provides tools for ingesting sample data and visualizing it through Kibana.

Note: For ongoing data ingestion, Telegraf is recommended. This application handles the initial setup and sample data ingestion.

Setup Instructions
1. Setup Elasticsearch
python setup_elasticsearch.py --host http://localhost:9200 --wait

This will create ingest pipelines, index templates, and data streams in Elasticsearch.

2. Generate Sample Data
python sample_data_generator.py --count 1000 --interval 15

This will generate sample temperature and air quality data in CSV format.

3. Ingest Sample Data
python ingest_bulk_to_elasticsearch.py --csv temperaturesensor_data.csv --index temperaturesensor-ds --host http://localhost:9200
python ingest_bulk_to_elasticsearch.py --csv airqualitysensor_data.csv --index airqualitysensor-ds --host http://localhost:9200

This will ingest the sample data into the Elasticsearch data streams.

4. Setup Kibana
python kibana_setup.py --kibana-host http://localhost:5601 --es-host http://localhost:9200 --wait

This will create index patterns and sample visualizations in Kibana.
